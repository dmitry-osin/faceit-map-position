# Faceit Map Position

A Chrome extension that automatically sends your preferred position to the Faceit chat based on the current map.

## How it works

1. Configure your preferred positions for each map in the extension popup
2. When a Faceit match starts, the extension detects the map
3. It automatically connects to the chat and sends your position

## Features

- Configurable map-position mappings
- Light / Dark / Neon themes
- English and Russian language support
- Open source, no viruses, keyloggers, or miners

## Installation

1. Open `chrome://extensions`
2. Enable "Developer mode"
3. Click "Load unpacked"
4. Select the `faceit-map-position` folder
